# Pagina web La 45-Style

Pagina web desarrollada por MrChamarravi-Dev para el emprendimiento de Modding La 45 Style, esta pagina web esta diseñada para mostrar los productos y servicios que ofrece el emprendimiento, ademas de contar con un sistema de carrito de compras y un sistema de pago integrado.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Git y GitHub
- React js
- FireBase
